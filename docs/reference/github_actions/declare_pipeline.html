Declare Pipeline¶

Declares a pipeline in Merkely

Invocation¶

docker run \
    --env MERKELY_COMMAND=declare_pipeline \
    --env MERKELY_API_TOKEN="${MERKELY_API_TOKEN}" \
    --env MERKELY_HOST="${MERKELY_HOST}" \
    --rm \
    --volume ${YOUR_MERKELY_PIPE}:/Merkelypipe.json \
    merkely/change

Parameters¶

NAME

Required?

Notes

MERKELY_COMMAND

yes

The Merkely command to execute.

MERKELY_API_TOKEN

yes

Your API token for Merkely.

MERKELY_HOST

no

Defaults to https://app.compliancedb.com
